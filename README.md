# 🏠 Housaty

> My very first web development project.

Housaty is a simple responsive landing page for a fictional apartment rental platform. Although it was built while I was learning HTML, CSS, and JavaScript, it represents the beginning of my journey into software engineering.

## ✨ Features

- Responsive landing page
- Mobile navigation menu
- Property listing cards
- Interactive buttons
- Clean beginner-friendly UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript


## 🎯 Purpose

This project was created for learning purposes and marks my first experience building a complete website from scratch.

Keeping it here reminds me how far I've come—from a simple landing page to building SaaS platforms, cybersecurity tools, and AI-powered applications.
